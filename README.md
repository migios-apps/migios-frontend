# Migios - Gym Membership Management

Aplikasi manajemen membership gym premium dengan React, TypeScript, dan Shadcn UI.

## Setup

### Instalasi Dependencies

```bash
npm install
```

### Menjalankan Development Server

```bash
npm run dev
```

Aplikasi akan berjalan di `http://localhost:5173`

## Build

### Development

```bash
npm run build:dev
```

### Staging

```bash
npm run build:staging
```

### Production

```bash
npm run build:prod
```

## Scripts Lainnya

- `npm run lint` - Cek kode dengan ESLint
- `npm run lint:fix` - Perbaiki error ESLint otomatis
- `npm run prettier` - Cek format kode
- `npm run prettier:fix` - Format kode otomatis
- `npm run preview` - Preview build production
